# AI Student Guide Hub

A comprehensive platform mapping AI mock interviews, interactive roadmaps, and student guides to assist students.

## Project Structure

This project is a Monorepo containing:
- `client/`: React / Vite frontend
- `server/`: Node.js / Express backend

## Getting Started

1. Clone the repository
2. Install dependencies in both `client` and `server` folders
3. Set up `.env` files based on `.env.example`
4. Run `npm run dev` in `client` and `node index.js` in `server`
